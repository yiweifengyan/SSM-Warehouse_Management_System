# 超市仓库管理系统 By Shaun and Arthur.  
基于SSM（Spring+SpringMVC+Java List，无数据库）开发的超市仓库管理系统，小巧精悍，界面做了美工，各种排错处理都有，能满足普通的本科毕业设计要求。  
后端源代码均在src文件夹内，前段网页代码均在web-content文件夹内，各种使用到的库已经包含在里面了。
